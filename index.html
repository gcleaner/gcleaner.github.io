Clean Your GNU/Linux System with Style

GCleaner is an open-source system cleaner that helps you maintain your GNU/Linux system efficiently. Join our community of developers and contribute to this amazing project!

Get Started Contribute

Why Choose GCleaner?

Open Source

Built with Python 3, GCleaner is completely open source and welcomes contributions from developers worldwide.

Light & Fast

Optimized for performance, GCleaner provides a smooth and efficient cleaning experience for your system.

Beautiful UI

Built with GTK+ 4.0, GCleaner offers a modern and intuitive interface that feels right at home on your Linux desktop.

Quick Start


# Clone the repository
git clone https://github.com/gcleaner/gcleaner-python.git

# Install dependencies (Fedora)
sudo dnf install gtk4-devel python3-gobject-devel python3-pytest polkit dmidecode

# Install dependencies (Ubuntu)
sudo apt-get install libgtk-4-dev python-gi-dev python3-pytest pkexec dmidecode

# Run the application
python3 gcleaner/application.py

For complete installation instructions including GLib schema and Polkit setup, visit our GitHub repository.
