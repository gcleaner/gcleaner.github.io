Open Source, Light and Dynamic

Open Source, Light and Dynamic

GCleaner Allows to keep clean your GNU/Linux system easily. Is so light and aerodynamically simple as a feather. Its source code is freely available for anyone to download and modify, leading to active development and a diverse feature set. There are currently two versions of GCleaner alive, one written in Vala and the other in Python, allowing any type of developer to participate in the project. Help us to make this project a reality.





GTK+

GCleaner handles the latest technologies such as GTK+ 3.0 toolkit to provide a clean and innovative user interface.

Vala

Vala programming language that is so lightweight has been stimulated by Elemenary OS and GNOME to gives GCleaner an incredible speed.

Python 3

Python 3 programming language has gained its place to promote variety within the development of GCleaner and include amazing features that only python can reach.
